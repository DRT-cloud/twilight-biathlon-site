How to use this editor:
1. Make your changes in the fields below.
2. Click Generate Updated CONFIG at the bottom.
3. Copy the generated code block.
4. Open twilight-biathlon.html in a text editor (Notepad works fine).
5. Find the section that starts with <script id="site-config"> and ends with </script>.
6. Replace that entire block with the copied code.
7. Save the file and upload it to Netlify — done.
